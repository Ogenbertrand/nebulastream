# Development

**Prerequisites**
- Node.js >= 20.11.0
- npm (bundled with Node.js)

**Install and build**
```bash
npm install
npm run build
```

**Run the API service**
```bash
npm run --workspace @nebula/api start
```

**Run the console app**
```bash
npm run --workspace @nebula/console start
```

**Clean build artifacts**
```bash
npm run clean
```
